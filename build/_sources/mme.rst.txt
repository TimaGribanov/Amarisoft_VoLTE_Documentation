Параметры конфига
-----------------
Синтаксис конфигов
~~~~~~~~~~~~~~~~~~
Оформление конфигурационных файлов схоже по структуре с JSON.

1. Поддерживаемые типы данных:

    * Числа (64-битные с плавающей точкой): `13.4`;
    * Комплексные числа: `1.2+3*I`;
    * Строки: `"string"`;
    * Булеаны: `true` или `false`;
    * Объекты: `{ field1: value1, field2: vaue2, ... }`;
    * Массивы: `[ value1, value2, ... ]`;
    
2. Базовые операции +, -, * и / поддерживаются с числами и комплексными числами. + также соединяет строки. Также поддерживаются операторы !, ||, &&, ==, !=, <, <=, >=, >.
3. Числа 0 и 1 поддерживаются как синонимы `false` и `true`.
4. {} на верхнем уровне опциональны.
5. ” для имён значений опционально.
6. Значения могут быть продублированы. Перезапись будет проихводится соответственно порядку чтения файла.
    
    .. code-block:: javascript
        
        {
            value: "foo",
            value: "bar",
            sub: {
                value: "foo",
            },
            sub: {
                value: "bar",
            }
        }
    
    Равнозначно
    
    .. code-block:: javascript
        
        {
            value: "bar",
            sub: {
                value: "bar"
            }
        }

7. Возможно добавление файлов командой `include`. Массивы добавлять нельзя.
    Если файл *file1.cfg* выглядит так:
    
    .. code-block:: javascript
        
        value: "foo",
        include "file2.cfg",
        foo: "foo"
    
    А файл *file2.cfg* так:
    
    .. code-block:: javascript
        
        value: "bar",
        foo: "bar"
    
    То конечные параметры будут:
    
    .. code-block:: javascript
        
        {
            value: "bar",
            foo: "foo"
        }

Элементы
--------
MME взаимодействует со следующими элементами

Из LTE
~~~~~~
UE
eNB
S-GW
MSC-сервером/VLR
HSS

Не из LTE
~~~~~~~~~
Сервером лицензирования Amarisoft (для гибких [float] лицензий).
Сервером дял удалённого управления WebSocket.

Таймеры, используемые в amarisoft
---------------------------------
+-------------------------+--------------------------------------+--------------+
| Обозначение таймера     | Назначение таймера                   | Значение     |
|                         |                                      | по умолчанию |
+=========================+======================================+==============+
| t3402                   | Значение таймера T3402 (в секундах). | -1           |
|                         | Значение -1 (по умолчанию) означает, |              |
|                         | что значение таймера не передаётся   |              |
|                         | в attach accept или TAU accept,      |              |
|                         | поэтому UE использует своё значение  |              |
|                         | по умолчанию (12 минут).             |              |
+-------------------------+--------------------------------------+--------------+
| t3412                   | Значение в секундах таймера T3412    | 1800         |
|                         | (TAU update). Значение -1 означает,  |              |
|                         | что таймер выключен. Его значение    |              |
|                         | отправляется на UE по протоколу NAS  |              |
|                         | до тех пор, пока UE не запрашивает   |              |
|                         | использование более долгого времени  |              |
|                         | ожидания (T3412 extended value       |              |
|                         | information element).                |              |
+-------------------------+--------------------------------------+--------------+
| t3412_low_priority      | Значение в секундах таймера T3412    |              |
|                         | (TAU update); используется, если UE  |              |
|                         | указывает низкий приоритет           |              |
|                         | NAS-сигнализации. Значение -1        |              |
|                         | означает, что таймер выключен. Его   |              |
|                         | значение передаётся на UE, пока UE   |              |
|                         | не запрашивает использование более   |              |
|                         | долгого времени ожидания (T3412      |              |
|                         | extended value information element). |              |
+-------------------------+--------------------------------------+--------------+
| t3412_extended_forced   | Значение в секундах таймера          | -1           |
|                         | T3412_extended, если UE использует   |              |
|                         | PSM. Если оно не равно -1, MME       |              |
|                         | проигнорирует запрошенное UE         |              |
|                         | значение и отправит вместо него вот  |              |
|                         | это.                                 |              |
+-------------------------+--------------------------------------+--------------+
| force_t3412_extended_ie | Если false, то MME выбирает          | false        |
|                         | наибольшее значение T3412            |              |
|                         | из установленного в MME              |              |
|                         | и запрошенного UE для PSM (пока      |              |
|                         | не установлено t3412_extended        |              |
|                         | _forced) и не отправляет T3412       |              |
|                         | _extended_ie, если значение может    |              |
|                         | быть закодировано как GPRS timer IE. |              |
|                         |                                      |              |
|                         | Если true, то MME принимает          |              |
|                         | запрошенное UE значение T3412        |              |
|                         | меньшим, чем установленное           |              |
|                         | в настройках, и всегда отправляет    |              |
|                         | T3421_extended_IE.                   |              |
+-------------------------+--------------------------------------+--------------+
| t3324_forced            | Значение таймера T3324 (в секундах), | -1           |
|                         | если UE использует PSM. Если         |              |
|                         | отличается от -1, то MME             |              |
|                         | проигнорирует запрошенное UE         |              |
|                         | значение и отправит вместо него      |              |
|                         | установленное в этом параметре.      |              |
+-------------------------+--------------------------------------+--------------+
| t3346                   | Значение таймера T3346 (в секундах). | -1           |
|                         | Передаётся в сообщении reject, если  |              |
|                         | причина EMM — #22 (перегрузка,       |              |
|                         | congestion) и значение не -1.        |              |
+-------------------------+--------------------------------------+--------------+
| t3448                   | Значение таймера T3448 (в секундах). | -1           |
|                         | Передаётся, если значение не -1 и UE |              |
|                         | указывает на поддержку таймера в UE  |              |
|                         | network capability information       |              |
|                         | element.                             |              |
+-------------------------+--------------------------------------+--------------+
| t3460                   | Значение таймера T3460 (в секундах). | 6            |
+-------------------------+--------------------------------------+--------------+
| t3460_wb_s1_ce          | Значение таймера T3460 (в секундах)  | 24           |
|                         | для UE, работающих в режиме          |              |
|                         | WB-S1/CE.                            |              |
+-------------------------+--------------------------------------+--------------+
| t3396                   | Значение таймера T3396 (в секундах). | -1           |
|                         | Таймер передаётся в сообщениях ESM   |              |
|                         | reject, если значение не -1.         |              |
+-------------------------+--------------------------------------+--------------+

Про таймеры, определённые 3GPP, подробнее на странице "Таймеры".
